Segmentación de usuarios

Objetivo

Obtener perfiles de usuarios finales que sirvan a todas las áreas para poder hacer envío de comunicación efectiva a través de un proceso que se describe a continuación.

Primer acercamiento del proyecto

Se contaba con una investigación previa de una agencia donde se establecían 36 perfiles, todos estos con características similares, así que el objetivo era poder reducir a una cantidad considerable y con las características indispensables para cada área que usará estos perfiles.

Entrevistas

Después de analizar la información se continúa a realizar entrevistas a las diversas áreas que realizaban envío de información a los usuarios. Fue importante conocer la metodología que usaban para este envío, en ese momento se detecta que no existía un orden entre áreas para este envío de información. En estas entrevistas también cada uno de ellos expreso las necesidades que se debían cubrir con esta segmentación y el como les gustaría trabajar con las demás áreas involucradas.

Desarrollo

Al tener la información recopilada se comienza a trabajar con los datos encontrados tanto de entrevistas como de la investigación anterior, se tiene reuniones semanales con los involucrados para que den feedback de los segmentos que se van creando semana a semana para que con toda esta nueva información se vaya reduciendo la cantidad de segmentos y de información obtenida.

Segmentación final

Después de hacer varias sesiones de trabajo con las áreas involucradas y de ir mejorando las características que cada uno necesitaba, pudimos terminar con 4 perfiles con todos los requerimeintos de cada área para el envío de información hacia usuarios finales

Mapa de envíos

Ya con los segmentos definidos se pudo proponer un mapa de envíos entre las áreas para que el usuario no recibiera información duplicada o que no era de valor para él, con todo esto se redujo un 18% las llamadas a call center con quejas de los clientes para solicitar su baja del envío de correos y se aumentó la apertura de estos.
